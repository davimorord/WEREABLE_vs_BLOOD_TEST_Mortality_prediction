# WEREABLE_vs_BLOOD_TEST_Mortality_prediction
Can a wrist-worn accelerometer replace a blood test for predicting 10-year mortality? XGBoost models using NHANES 2003-2004 data (n=2,755, 216 events). Wearable-only model achieves AUC 0.828 vs blood test AUC 0.822. Includes feature engineering, model training, threshold analysis, and clinical interpretation.
